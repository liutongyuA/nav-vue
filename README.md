## nav-vue 使用vue搭建的前端导航站点

### 安装与使用
```
npm install
npm run serve
```

### 项目说明
前端导航项目可以帮助你快速搜素内容、添加常用网站。它也兼容移动端，为我们上网提供更多便捷！

### 构建背景
开始我是使用JQuery实现这个项目，当我学习vue后，重新做了这个项目，项目里也用到了我自己封装的UI组件，如：button、input、dialog、message等等。
当然项目目前也还有很多不足，我会慢慢深入学习，努力写出更优秀的代码！:muscle:
